Horror Game Maker 01 
created by VIS-Games
--------------------
This package contains an example 
project to create an SCP-like Horror 
Game including a jumpscare at the end.

Feel free to use it for learning and 
building your own small games like
this.

All included content can be used in 
your own projects.

-----------------------------------------
Use WASD to move and mouse to turn camera.
Use F to use the flashlight.
